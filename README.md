# Meeting Room Booking App

A simple full-stack application for booking meeting rooms.

## Tech Stack
- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB with Mongoose

## Setup

### Backend
1. Navigate to `backend` directory
2. Run `npm install`
3. Create a `.env` file with `MONGO_URI=mongodb://localhost:27017/meeting-room-db`
4. Run `npm run dev`

### Frontend
1. Navigate to `frontend` directory
2. Run `npm install`
3. Run `npm start`

## Features
- Basic structure for meeting room booking
